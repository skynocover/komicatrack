## How to use 

1. 將網址貼上"在這裡輸入網址"後按下新增按鈕
2. 若網址符合規則則顯示在下面
3. 按下前往會自動開啟瀏覽器,按下刪除則會刪除追蹤的討論串
- 可以使用除了Chrome以外的瀏覽器,請按下瀏覽器設定的按鈕
